Fortnite

Fortnite is an online game developed by Epic Games in the form of different software packages offering different game modes that share the same general gameplay and the same game engine. The game modes include Fortnite: Save the world, a cooperative shooting game and survival game designed for a maximum of four players and whose goal is to fight hordes of zombies and defend objects with fortifications to be built by the player (s), and Fortnite Battle Royale, a battle royale game formerly paying, now in free-to-play where up to 100 players fight in increasingly smaller spaces to finish as the last person standing.
